# HEARTBEAT.md

## Periodic Checks

When heartbeat fires, check these in order:

1. **Memory Maintenance** — Review recent `memory/YYYY-MM-DD.md` files. Update `MEMORY.md` with significant decisions or learnings worth keeping long-term.

2. **Open Questions** — Check if any items in `MEMORY.md > Open Questions` can be updated (e.g., regulatory news, API changes).

3. **NPHIES Sandbox Status** — If sandbox credentials are available, note any API deprecation or version change notices from NPHIES portal.

4. **Project Status** — If there are recent commits or active development, summarize what changed and flag any blockers.

If nothing needs attention: `HEARTBEAT_OK`
