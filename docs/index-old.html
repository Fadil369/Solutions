Deployment

One-Click Deploy

Configure two GitHub secrets, then every push to main auto-deploys Pages and Workers.

  • Fork & Clone the Repository

    Fork Fadil369/Solutions and clone it locally.

  • Add GitHub Secrets

    In your repo → Settings → Secrets and variables → Actions, add: CF_API_TOKEN (Cloudflare API token with Workers + KV + D1 + R2 edit permissions) and CF_ACCOUNT_ID (your Cloudflare account ID).

  • Create Cloudflare Resources

    Run npm run cf:setup from the workers/ directory. This creates the KV namespaces, D1 database, and R2 bucket via Wrangler, then patches wrangler.toml IDs automatically.

  • Push to Main

    Push any change to main. The deploy-pages and deploy-workers workflows run in parallel — Pages deploys in ~30s, Workers in ~60s.

  • Enable GitHub Pages

    In your repo → Settings → Pages, set Source to GitHub Actions. Your site will be live at https://your-username.github.io/Solutions/.
